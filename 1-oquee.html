O que é HTML?

O HTML é uma linguagem de marcação desenvolvida pelo britânico Tim Berner. O HTML é uma linguagem utilizada descrever documentos utilizando tags, sendo composto pela seguinte estrutura básica:

  • !DOCTYPE: especifica o tipo de documento que contém dentro do arquivo, sendo especificado no ínico da página HTML.
  • html: seria a tag geral onde todo o arquivo HTML acontece.
  • head: é o cabeçlho da página
  • title: representa o título da página
  • body: é o corpo do documento, onde desenvolve a estrutura e o conteúdo do seu documento.

De forma geral, o HTML é a linguagem que define os elementos gráficos de uma página web.
