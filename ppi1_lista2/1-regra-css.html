O que é uma regra CSS?

Uma regra CSS é o que define e inclui estilo e posição nos arquivos HTML. É composta por um seletor, uma propriedade e um valor.

  1. Seletor: Seleciona o elemento HTML que será estilizado por aquela regra
  2. Propriedade: parte da regra onde você define o que vai adiconar de estilo
  3. Valor: onde você define a forma que vai ser estlizado pela propriedade

Exemplo: h1{color: purple;}
