O que uma regra CSS?

CSS é um um arquivo chamado folha de estilo, sendo onde definimos o estilo dos elementos HTML. o CSS é definido regras, as quais são definidas da seguinte forma:

  • Seletor: indica a tag/elemento em que a seguinte regra será utilizada.
  • Declaração: é onde você define a regra/aplica a aparência. Sendo dividida em propriedade e valor.

Exemplos

        ul{/* -> Seletor*/
            background-color: #e17beb; /* -> Declaração*/
            border: groove #a02cff;
            /* ^ propriedade*//* ^ Valor*/
            color: #00028d;
        }
