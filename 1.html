Comandos Git

  • git init - cria um repositório em um diretório
  • git add - adiciona modificações d workcopy/ worktree ai stage
  • git commit -m "asd" - cria um commit apartir do stage
  • git log - exibe os commits do nosso repositório local
  • git log --oneline --all --graph
  • git status - exibe o estado do WK
  • git checkout -b nova-branch - cria uma nova branch apontando para a branch ou hash informado
  • git reset --hard - restaura os arquivos trakeados ao estado no HEAD
  • git push remote branch
  • git config user.name = "nome"
  • git config user.email = "email"
  • git clone url - faz o clone de um repositório
